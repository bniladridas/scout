Scout

Find heavy files, caches, build artifacts, models, and toolchains. Move only what you choose to Trash.

Download DMG Version

Install

Open the DMG, drag Scout to Applications, then launch it from Applications.

What it does

Scans your home folder or any folder you choose.
Labels each item before you act.
Uses macOS Trash. No permanent delete.

Build

With Xcode command line tools installed:

scripts/package-dmg.sh

Scout is local and has no network feature. It is not Apple-notarized, so macOS may warn before opening it.
